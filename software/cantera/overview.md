# Cantera 能力定位

Cantera 是面向热力学、化学动力学、输运性质和反应流建模的开源软件。它适合把化学反应机理、火焰结构、反应器网络和机理分析转化为可复现 benchmark。

## 主要适用能力

- 热力学状态、平衡计算和混合物性质计算。
- 化学动力学积分和点火延迟分析。
- 输运性质计算和反应扩散耦合。
- 0D 反应器、反应器网络和理想反应器模型。
- 一维自由传播预混火焰、对向扩散火焰和受拉伸火焰。
- 反应路径分析、敏感性分析、机理简化和机理对比。

## 能力边界

Cantera 不是通用三维 CFD 主求解器。它可以提供详细化学、热力学、输运、0D/1D 反应流 benchmark，也可以作为 CFD 或多物理场 workflow 的化学动力学后端，但不应被包装成通用三维流场求解器。

## 能力注册层中的角色

在个人 workflow 或 MatOS 中，Cantera 应优先承担燃烧和反应动力学 capability 的 benchmark 生成、物理趋势验证、机理分析和轻量自动验收角色。
