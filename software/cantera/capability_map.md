# Cantera 能力地图

| 编号 | 能力名称 | 问题类型 | 资产化目标 |
| --- | --- | --- | --- |
| C01 | constant-pressure ignition | 0D 定压点火 | 输入燃料/氧化剂/温度/压力/机理，输出点火延迟、温度曲线和关键组分曲线。 |
| C02 | freely propagating premixed flame | 一维自由传播预混火焰 | 输入预混气、压力、温度和网格策略，输出火焰速度、温度和组分剖面。 |
| C03 | counterflow diffusion flame | 一维对向扩散火焰 | 输入燃料/氧化剂入口、质量通量、间距和辐射模式，输出火焰结构、峰值温度和火焰位置。 |
| C04 | extinction strain rate | 扩散火焰熄灭极限 | 扫描拉伸率或质量通量，输出熄灭边界、收敛状态和临界趋势。 |
| C05 | reaction path analysis | 反应路径分析 | 输入机理、状态点和目标元素，输出反应路径图和主导反应链。 |
| C06 | mechanism reduction | 机理简化 | 输入详细机理、工况集合和误差阈值，输出简化机理和误差评估。 |
